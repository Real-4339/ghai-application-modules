# Start an application with Maven

```
mvn clean install
```

```
cd server
mvn compile exec:java --% -Dexec.mainClass="fiit.vava.server.Server"
```

```
cd client
mvn install org.openjfx:javafx-maven-plugin:0.0.8:run 
```

# Compile to .jar

```
cd server
mvn clean compile package
```

```
cd client
mvn clean package
```

.jar s are in target folders.
